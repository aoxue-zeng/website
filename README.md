# 月蚀·玫瑰残响

一款剧情向像素网页游戏。主角 **艾瑟·塔柳斯·卡瑞娜**——艾瑟公爵、卡瑞娜魔女之子、月·浪漫·战争之神——被卷入「副本系统」，在副本游戏式的世界里逐步揭开系统与双世界的真相。

## 玩法

- **视觉小说对话**：像素立绘 + 打字机文字框 + 分支选项
- **副本系统**：进入副本 → 系统发布任务/规则 → 探索/战斗/解谜 → 通关或失败。3 章主线副本（迷雾森林 → 古堡迷踪 → 时空塔），3 种结局
- **回合制战斗**：攻击 / 技能 / 防御 / 道具
- **休闲系统**：农场种植循环（种→浇水→成熟→收获→出售）、公爵府房屋装修（网格化摆放家具，影响舒适度）
- **小镇日常**：NPC 委托副线 + 好感度
- **存档**：localStorage 3 存档位

## 操作

- 鼠标点击 / `Enter` / `Space`：推进对话
- `↑↓` 或 `W/S`：选择选项
- 探索时点击行动按钮

## 角色

- **艾瑟·塔柳斯·卡瑞娜**：三重身份的主角。左眼是玫瑰（暗红），齐肩黑发，黑色骑士装，偏好暗红与黑。
- **薇拉·安瑟尔**：管家。看上去 20+，实际实力极强，自塔柳斯幼时起照料她，忠心但不约束。

## 技术架构

纯前端（HTML + CSS + 原生 JS + Canvas），无构建依赖，可直接托管于 GitHub Pages。

```
├── index.html
├── css/   (style.css, pixel.css)
├── js/
│   ├── main.js
│   ├── core/    (game, dialogue, scene, sprite, input, save, system-panel)
│   ├── systems/ (combat, dungeon, farm, house, inventory, quest)
│   └── data/    (story, dungeons, items, npcs, sprites)
└── README.md
```

数据与逻辑分离——剧情 / 物品 / NPC / 副本全部以数据形式存于 `js/data/`，便于后续扩展章节与内容。

## 本地预览

用任意静态服务器打开即可，例如：

```bash
python -m http.server 8000
# 浏览器访问 http://localhost:8000
```
